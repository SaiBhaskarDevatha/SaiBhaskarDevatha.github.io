Abstract

This primary aim of the project is to build a Music classfying based on its genere.The system includes feature extractor which extracts temporal features and frequency features to genrate a feature vector.The feature vector is sent through a classfier, trained over GTZAN Dataset, which is considered as standard dataset for the genere classification. The classifier classifes the frame into Classical,Pop,Metal,Rock generes.Pitch,Spectral centroid,MFCC,ZCR are being computed for feature vector.
